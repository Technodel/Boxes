Technodel Boxes

Laptop inventory tracking system organized by storage box.

Features

  • Create and manage named storage boxes, each holding multiple laptops.
  • Add laptops individually or in bulk (paste a list of names).
  • Drag-and-drop laptops between boxes with a logged move comment.
  • Mark any laptop as SOLD — it is removed and the sold counter updates.
  • Toggle laptop status between "Ready" and "Not Ready" for quick prep checks.
  • Global search (Ctrl+K) to instantly find any laptop across all boxes.
  • Full activity history log with timestamps, filterable and clearable.
  • State syncs to the local dashboard server and falls back to localStorage.
